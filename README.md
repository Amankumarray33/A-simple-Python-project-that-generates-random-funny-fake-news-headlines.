# Fake News Headline Generator

This is a fun beginner-friendly Python project that generates random fake breaking news headlines using the `random` module.

## Features
- Generates random funny headlines
- Uses Python lists and `random.choice()`
- Runs in a loop until the user exits
- Beginner-friendly project

## Example Output
BREAKING NEWS: Aman dances with a Plate of samosa  
BREAKING NEWS: A Mumbai cat declares war on at India Gate

## Technologies Used
- Python
- Random module
